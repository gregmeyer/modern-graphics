
- **Expected**: Ample whitespace, simple color palette, clean fonts
- **Code**:
```python

template = quick_template_from_description(prompt)
register_template(template)
```

## Complete Examples with Code

### Example 1: Tech Startup Pitch Deck
```python
from modern_graphics import quick_template_from_description, register_template, ModernGraphicsGenerator
from pathlib import Path

prompt = "bold vibrant colors with blue and purple accents, tech startup aesthetic, clean modern sans-serif font, energetic and innovative feel, high contrast for impact"

template = quick_template_from_description(prompt)
register_template(template)

generator = ModernGraphicsGenerator("Product Vision", template=template)
html = generator.generate_story_slide(
    title="AI-Powered Future",
    what_changed="Manual processes â†’ Automated intelligence",
    time_period="2024-2025",
    what_it_means="10x efficiency improvement"
)
generator.export_to_png(html, Path("pitch_slide.png"))
```

### Example 2: Corporate Report
```python
prompt = "corporate blue and gray, professional, traditional fonts, conservative and trustworthy"

template = quick_template_from_description(prompt)
register_template(template)

generator = ModernGraphicsGenerator("Q4 Report", template=template)
html = generator.generate_comparison_diagram(
    left_column=[{'title': 'Q3', 'items': ['Metric 1', 'Metric 2']}],
    right_column=[{'title': 'Q4', 'items': ['Improved 1', 'Improved 2']}]
)
generator.export_to_png(html, Path("corporate_comparison.png"))
```

### Example 3: Creative Portfolio
```python
prompt = "creative and artistic, unique color palette, expressive typography, bold and distinctive"

template = quick_template_from_description(prompt)
register_template(template)

generator = ModernGraphicsGenerator("Portfolio", template=template)
html = generator.generate_slide_card_diagram(
    cards=[
        {'title': 'Project 1', 'subtitle': '2024', 'description': 'Creative work', 'color_key': 'blue'},
        {'title': 'Project 2', 'subtitle': '2024', 'description': 'Artistic piece', 'color_key': 'purple'}
    ]
)
generator.export_to_png(html, Path("portfolio_cards.png"))
```
